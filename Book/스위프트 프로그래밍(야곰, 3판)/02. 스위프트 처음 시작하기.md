## 2.2 Console Log

print() : 출력하려는 인스턴스의 description 프로퍼티에 해당하는 내용 출력  
dump() : 출력하려는 인스턴스의 자세한 내부 콘텐츠까지 출력  
문자열 보간법 - `"\(변수 또는 상수)"`

→ Markup Formatting Reference

중첩 주석

```swift
/* 여러 줄 주석 안 쪽에

*/

*/
```
